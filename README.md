
# Memory Game

This Project Let You Choose From 3 Main Catgories To Play With Cards Of The Category


## Features

- 3 Gamemode Button Selctor
- Custom Backs And Fronts For Each Category
- Play Again Option


## Gamemodes
-   Cats Gamemode
-   Dogs Gamemode
-   Harry Poter Gamemode
